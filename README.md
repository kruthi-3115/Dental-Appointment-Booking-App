Project Title: Dental Appointment Booking App

Objective:
This project is a mobile UI/UX design for a dental clinic appointment booking system. It helps users book appointments easily without phone calls or manual scheduling.

Features:
- Login / Signup screen
- Service selection (Dental Checkup, Whitening, Braces)
- Doctor selection
- Date & time slot booking
- Appointment confirmation screen

Tools Used:
- Figma (UI Design)

Prototype:
A clickable prototype is created to show the full booking flow.
